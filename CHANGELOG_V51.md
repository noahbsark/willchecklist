# V51 Changelog — Setup Flow Simplification

V51 continues the simplicity work by reducing Step 1 and making the setup wizard easier to understand.

## Changes

- Reduced Step 1 from four questions to three.
- Removed the “who are these notes for?” question because it did not change the user’s next step.
- Kept the safety check as the first question.
- Reworded setup questions to be shorter and more concrete.
- Updated the progress indicator so Question 1 of 3 starts at one-third progress instead of appearing empty.
- Simplified the setup result summary so it only reflects choices that matter to the flow.
- Added focus/scroll behavior after Next/Previous so the next question is easier to follow.

## Simplicity principle

Do not ask questions unless the answer changes the experience or prevents misuse.
