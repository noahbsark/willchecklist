# Changelog V25 — Plain-Flow Simplicity Pass

This version focuses on reducing reading, reducing repeated safety language, and making the six-step core path easier to follow.

## What changed

- Added the `v25-plain-flow` core-page style to the six main steps.
- Shortened the bottom action buttons from longer "Save and go..." labels to simple "Next: ..." labels.
- Kept save behavior intact: clicking the main Next button still saves the step in browser-local storage.
- Replaced repeated instruction cards with one short safety line on core pages.
- Reduced repeated warning boxes on core pages.
- Made the mobile step indicator smaller and less cluttered.
- Simplified homepage wording.
- Added cleaner progress-page styling.

## Core public path

1. Setup
2. Checklist
3. Documents
4. Beneficiaries
5. Summary
6. Final Review

## Public launch note

This ZIP is still suitable for public-test mode. Newsletter, lead, ad, and affiliate areas should remain hidden or paused until real endpoints and approved programs are connected.
