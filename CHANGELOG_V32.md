# Changelog V32

Focus: make the six-step flow feel less clickable and more guided.

## Changed

- Converted the six-step strip on the core pages from a secondary navigation menu into a simple progress indicator.
- Kept movement through the flow on the single bottom action button.
- Added progress-strip styling for completed, current, and upcoming steps.
- Updated progress JavaScript so completed checkmarks still work with the non-clickable strip.
- Kept the public ZIP limited to the `/site` folder only.

## Why

Users should not feel like they need to choose among six steps. The page should show where they are, then give one clear next action.
