# Changelog V37

Simplicity cleanup focused on the Continue and Progress pages.

- Continue now opens with one plain heading and one primary next-step button.
- Removed duplicate quick-start strips, decorative hero images, and extra path sections from Continue and Progress.
- Progress now shows one recommended action first, then the six-step list.
- Core utility drawers now say "Print, copy, or clear" instead of a vague "More options."
- Optional notes headings were softened from "Private planning notes" to "Optional notes" on core pages.

No new tools were added. The update is intentionally a reduction pass.
