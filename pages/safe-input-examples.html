Misuse prevention

Type general labels. Do not type secrets.

Use these examples anywhere the site asks for notes, locations, contacts, account categories, or planning questions.

Return to guided flowHow to use the site
SituationSafe to typeDo not type
Banking“Checking account at my bank — verify beneficiary if available.”Full account number, balance, routing number, login, password.
Life insurance“Life insurance policy — paper copy in blue folder.”Policy number, SSN, online account password, private ID numbers.
Document storage“Fireproof box in office closet.”Safe combination, alarm code, key location that creates security risk.
Digital accounts“Password manager exists; trusted person knows where instructions are.”Master password, private keys, recovery phrase, 2FA backup codes.
Family notes“Discuss guardian role with Alex.”Detailed private conflict, accusations, confidential legal strategy.
Professional questions“Ask whether beneficiary designations conflict with estate plan.”Full financial inventory with account numbers or sensitive identifiers.
Good pattern

Label + location

“Mortgage documents — home office file cabinet.”

Good pattern

Question + category

“Ask attorney whether retirement beneficiary designations need review.”

Bad pattern

Secret + access

“Password is ____” or “safe code is ____.” Do not store that here.

Simple rule

If the information could let someone access money, accounts, identity documents, private keys, email, or a locked space, do not type it into the site. Write where the trusted person can find your secure instructions instead.
