# Changelog V24 - Simplified Core Experience

This version focuses on fewer repeated instructions and fewer visible decisions on the six core pages.

## Changed
- Hid the duplicate hero area on the six core step pages so users see one clear step header.
- Reduced core-page navigation to Progress and Help; the page itself provides the next action.
- Shortened the global safety notice.
- Rewrote core-page legal warnings into a calmer one-line educational note.
- Kept print/copy/reset tools inside collapsible drawers.
- Made the footer compact on core pages to avoid pulling users away from the path.

## Core goal
A first-time user should understand: do the current step, save, then move to the next step.
