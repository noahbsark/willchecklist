# Changelog V21 — Simpler Core Flow

## What changed
- Made the end of each core step easier: the main next button now saves the current step before moving forward.
- Simplified the six core pages by hiding optional FAQ/extra sections from the main path.
- Added a small "Need help with this step?" expandable panel instead of showing extra help by default.
- Reduced visual weight on steppers, safety notes, cards, and utility drawers.
- Shortened homepage and progress page copy.

## Public-test note
This version keeps the visible experience focused on the six-step path. Optional and educational pages still exist, but they do not compete with the main user journey.
