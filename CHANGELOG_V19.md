# Changelog V19 - Direct Start and Simpler UI

## What changed

- Made the main **Start** navigation go directly to Step 1 instead of an overview page.
- Updated the homepage primary CTA to **Start Step 1**.
- Kept the overview page, but repositioned it as **How it works** rather than the required starting point.
- Simplified footer start links so users see Start Step 1, Continue, How it works, and Print guide.
- Shortened completion panel copy on the six core pages.
- Hid extra secondary buttons in the main recommended next-step panel to reduce choice overload.
- Added a calmer V19 CSS pass for core pages, completion panels, utility drawers, and mobile buttons.

## Public-test posture

V19 is still intended to be public-test friendly. Inactive monetization, ad, affiliate, newsletter, and lead paths should stay hidden or paused until connected.

## Publishing note

Upload only the `/site` folder. This V19 package is public-focused and does not require private builder notes.
