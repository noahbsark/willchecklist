# Changelog V23 — Ultra-Simple Core Flow

This update makes the public six-step experience calmer and easier to use.

## Changes

- Hid breadcrumbs on the six core step pages to reduce visual noise.
- Hid the floating Help button on core pages because each page already has local help.
- Tightened the top spacing and instruction cards on core pages.
- Added a mobile-only bottom Next button so phone users always have a clear next action.
- Kept the safety rule visible without adding more warnings.

## Core principle

The user should see one task, one safety reminder, and one next step.
