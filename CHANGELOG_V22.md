# Changelog V22 — One-Button Core Flow

This release keeps the site simple by reducing completion choices on the six core pages.

## Changed

- Replaced the separate “Mark done only” control on core step pages with a single primary “save and continue” action.
- Standardized the finish panel on all six core pages.
- Shortened core-page hero text.
- Simplified help drawers into one quiet “Need help?” section.
- Added V22 styling for calmer one-button finish panels.
- Added a small homepage note: “One step at a time. One main button per step.”

This is still in public-test mode. Inactive newsletter, lead, ad, and affiliate elements should remain hidden until connected.
